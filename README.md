PROJETO RESTAURANTE:

Fizemos uma espécie de menu de um restaurante, que pode ser acessado pelos funcionários para alterar os pratos e preços, ou pelos clientes para eles terem acesso ao cardápio composto por entradas, pratos principais e sobremesas. Também existe a opção de enviar uma reclamação ou sugestão.

O programa abre com uma interface que te da 4 opções sendo:
[1] Acesso a parte dos funcionários, podendo remover, adcionar e alterar o preço de um prato
[2] Acesso ao cardápio, com os menus de entrada, prato principal e sobremesa
[3] Abre a possibilidade do cliente enviar uma reclamação/sugestão
[4] encerra o programa
